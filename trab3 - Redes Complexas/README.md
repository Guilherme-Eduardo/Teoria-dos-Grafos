# Trabalho 3 - Análise de Redes Complexas


## Objetivo
O objetivo deste trabalho é analisar uma rede complexa.

## Especificação
A equipe deve escolher uma rede complexa disponível em algum banco de dados disponível na internet e aplicar diversas técnicas para analisar essa rede utilizando a ferramenta networkX e a linguagem Python. A rede pode provir de qualquer aplicação, mas sugere-se fortemente aplicações biológicas e/ou da saúde.

### Exemplos de técnicas que podem ser utilizadas para a análise da rede são:

- Ciclos, componente conexas, raio e diâmetro;
- Distribuição de graus, centralidade de vértices;
- Árvores Geradoras, clusters e cliques.
- Além da análise geral da rede, a seguinte pergunta deve ser respondida: qual modelo de grafo melhor representa a rede escolhida?

## Entrega
Este trabalho pode ser feito de forma individual ou em dupla e deve ser entregue um arquivo compactado contendo os seguintes arquivos:

relatorio.pdf, contendo a descrição da rede utilizada, link para acesso, desenho da rede (se viável), aplicação da qual ela foi extraída ou representa, e a análise em sim, com possíveis conclusões sobre a estrutura da rede, vértices importantes, etc.
<arquivos>.py com os códigos utilizados para analisar a rede.
A avaliação do trabalho será focada principalmente na qualidade e variedade das análises efetuadas.
